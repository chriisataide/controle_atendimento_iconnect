📱 O que é um PWA (Progressive Web App)?

PWA (Progressive Web App) é uma aplicação web que funciona como um aplicativo nativo. Você pode instalar nosso sistema diretamente no seu dispositivo e usar offline.

🚀 Vantagens do PWA:
  • Instalação rápida: Sem precisar da App Store
  • Funciona offline: Acesse dados mesmo sem internet
  • Notificações: Receba alertas importantes
  • Atualizações automáticas: Sempre na versão mais recente
  • Economia de espaço: Menor que apps nativos
📲 Como instalar:
Desktop (Chrome/Edge)

Procure o ícone de "instalar" na barra de endereços ou no menu ⋮

Mobile (Android/iOS)

Toque no menu do navegador e selecione "Adicionar à tela inicial"
