Siga o guia abaixo para instalar nosso sistema como um aplicativo em seu dispositivo:

💻 Desktop (Windows/Mac/Linux)
Google Chrome:
  1. Abra o site no Chrome
  2. Procure o ícone de "Instalar" na barra de endereços (à direita)
  3. Clique no ícone e selecione "Instalar"
  4. Confirme a instalação
Microsoft Edge:
  1. Clique nos três pontos (⋯) no canto superior direito
  2. Selecione "Aplicativos" → "Instalar este site como um aplicativo"
  3. Digite um nome para o app e clique em "Instalar"

O app aparecerá na sua área de trabalho e no menu iniciar

📱 Android
Chrome no Android:
  1. Abra o site no Chrome
  2. Toque nos três pontos (⋮) no canto superior direito
  3. Selecione "Adicionar à tela inicial"
  4. Confirme tocando em "Adicionar"
Firefox no Android:
  1. Toque nos três pontos no menu
  2. Selecione "Instalar" ou "Adicionar à tela inicial"
  3. Confirme a instalação

O ícone aparecerá na tela inicial do Android

🍎 iPhone/iPad (iOS)
Safari no iOS:
  1. Abra o site no Safari
  2. Toque no ícone Compartilhar (quadrado com seta para cima)
  3. Role para baixo e toque em "Adicionar à Tela de Início"
  4. Digite um nome e toque em "Adicionar"
Nota: No iOS, use preferencialmente o Safari para melhor compatibilidade.

O app funcionará como nativo no iOS

🛠️ Problemas Comuns
  • Certifique-se de que está usando um navegador compatível (Chrome, Edge, Safari)
  • Verifique se o site está sendo acessado via HTTPS
  • Tente recarregar a página (Ctrl+F5 ou Cmd+R)
  • Em alguns casos, aguarde alguns segundos após carregar a página
  • Aguarde alguns minutos após a primeira instalação
  • Use o app online algumas vezes antes de tentar usar offline
  • Verifique se o Service Worker está ativo nas ferramentas do desenvolvedor
