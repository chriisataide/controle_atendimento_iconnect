Guia de Instalação

Passo a passo para instalar o iConnect em cada plataforma

arrow_back Voltar ao PWA
desktop_windows
Desktop (Windows / Mac / Linux)

Chrome, Edge e outros navegadores baseados em Chromium

Google Chrome
  1. Abra o iConnect no Google Chrome
  2. Procure o ícone de "Instalar" (monitor com seta) na barra de endereços à direita
  3. Clique no ícone e confirme a instalação
  4. O app abrirá em uma janela própria e aparecerá na área de trabalho
Microsoft Edge
  1. Clique nos três pontos (⋯) no canto superior direito
  2. Selecione "Aplicativos" no menu
  3. Clique em "Instalar este site como um aplicativo"
  4. Digite um nome e clique em "Instalar"
android
Android

Chrome, Firefox e Samsung Internet

Chrome Android
  1. Abra o iConnect no Chrome do Android
  2. Toque nos três pontos (⋮) no canto superior direito
  3. Selecione "Adicionar à tela inicial"
  4. Confirme tocando em "Adicionar"
Firefox Android
  1. Abra o iConnect no Firefox
  2. Toque nos três pontos no menu
  3. Selecione "Instalar" ou "Adicionar à tela inicial"
  4. O ícone aparecerá na tela inicial do Android
phone_iphone
iPhone / iPad (iOS)

Apenas Safari suporta PWA no iOS

Safari iOS
  1. Abra o iConnect no Safari
  2. Toque no ícone Compartilhar (quadrado com seta para cima)
  3. Role para baixo e toque em "Adicionar à Tela de Início"
  4. Edite o nome se desejar e toque em "Adicionar"
info Importante: No iOS, use apenas o Safari para instalar PWAs. Chrome e Firefox no iOS não suportam essa funcionalidade.

helpProblemas Comuns

  • Certifique-se de que está usando um navegador compatível (Chrome, Edge, Safari)
  • Verifique se o site está sendo acessado via HTTPS
  • Tente recarregar a página (Ctrl+F5 ou Cmd+R)
  • Em alguns casos, aguarde alguns segundos após carregar a página

  • Aguarde alguns minutos após a primeira instalação para o cache carregar
  • Use o app online algumas vezes antes de tentar usar offline
  • Verifique se o Service Worker está ativo na página de Status do PWA

  • Verifique as configurações de notificação do navegador e do sistema operacional
  • No iOS, notificações push em PWAs são suportadas a partir do iOS 16.4+
  • Certifique-se de ter concedido permissão de notificações ao acessar o site
