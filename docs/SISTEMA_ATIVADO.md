## ⚙️ Configurações Avançadas

### Para Ativar APIs Externas:
1. **OpenAI (Chatbot):** Adicionar `OPENAI_API_KEY` no .env
2. **WhatsApp:** Configurar `WHATSAPP_TOKEN` no .env  
3. **Slack:** Definir `SLACK_WEBHOOK_URL` no .env
4. **Email:** Configurar SMTP no .env

### Para Produção:
1. Alterar `ENVIRONMENT=production` no .env
2. Configurar PostgreSQL
3. Configurar Redis para cache
4. Configurar Celery para tasks assíncronas
