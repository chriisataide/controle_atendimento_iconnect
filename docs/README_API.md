# Documentação das APIs iConnect

Acesse a documentação automática:
- Swagger: `/swagger/`
- Redoc: `/redoc/`

## Recursos disponíveis
- Tickets: criação, consulta, atualização, avaliação
- Clientes: consulta, abertura de chamados
- Integrações: WhatsApp, Slack, Webhooks
- Métricas, relatórios, automações

## Exemplos de uso
Veja exemplos práticos em `docs/API_EXEMPLOS.md`.

## Como ativar/usar
- APIs REST disponíveis por padrão (DRF)
- Para webhooks, consulte os endpoints em `dashboard/urls.py`
- Autenticação: Token ou sessão (ver Swagger)
